###Novedades de la versión 7.9.0.0

#####Actualización de funciones

>#####Se agregó una nueva opción para controlar si las carpetas de agregación de etiquetas deben mostrarse dentro del nodo de acceso rápido

>#####Se agregó una nueva opción para controlar si el visor incorporado se usa primero de forma predeterminada

>#####Se agregó soporte para Seer, que se puede usar para ver rápidamente el contenido del archivo

>#####Se agregó una nueva opción para controlar si los archivos o carpetas siempre se abren en una nueva pestaña

#####Mejora del rendimiento

>#####Se mejoró el diseño de bloqueos asíncronos y se mejoró el rendimiento en algunos escenarios

#####Corrección de errores

>#####Se corrigió un problema relacionado con el manejo de parámetros de arranque que resultaba en una pestaña en blanco adicional al abrir la ruta raíz de la unidad

>#####Se solucionó el problema de que algunos colores de texto pueden ser anormales al cambiar al modo Acrílico personalizado si el sistema usa un tema claro y la aplicación usa un tema oscuro

>#####Se arregló el raro caso de que el archivo de fuente especificado por el usuario causara un bloqueo si el archivo de fuente no está disponible o no se puede acceder a él. Si ocurre este problema, la aplicación volverá automáticamente a la fuente predeterminada

>#####Se solucionó el problema de que aparecía un cuadro de mensaje de error incluso si el acceso directo se creaba correctamente

>#####Se corrigieron algunos otros problemas que podrían causar fallas
