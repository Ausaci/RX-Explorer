###What's new in Ver.7.9.2.0

#####Bug Fixes

>#####Fixed the problem of not being able to drag and drop files inside RX-Explorer

>#####Fixed an issue that could allow the user to perform actions not allowed by the drag action initiator

>#####Fixed the problem that multiple identical files may be generated when dragging lnk files or url files from RX-Explorer to any location

>#####Fixed the problem that thumbnails will not be displayed when dragging a single lnk or url file

>#####Removed a mechanism preventing users from importing configuration files exported from other computers, and added a security risk warning for this
