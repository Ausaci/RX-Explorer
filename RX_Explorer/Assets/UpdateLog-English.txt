###What's new in Ver.8.0.4.0

#####Bug Fixes

>#####Fixed an issue that may make "Always on Top" setting invalid when opening .exe files

>#####Fixed an issue where transfer interruptions or connection refused in some cases might not be detected when sharing files via Bluetooth

>#####Fixed an issue where multiple selection of files in search results could cause a crash

>#####Fixed an issue where pressing the Esc key would not first close the windows of preview programs such as Quicklook and Seer

>#####Fixed an issue where Ftp files could not be opened with the built-in viewer

>#####Fixed an issue where Ftp files could not be copied or moved

>#####Fixed an issue where Windows Terminal was not automatically selected as the default terminal when it presents

>#####Fixed some other issues that could cause crashes
