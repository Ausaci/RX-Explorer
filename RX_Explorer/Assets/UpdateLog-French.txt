###Quoi de neuf dans la version 8.1.6.0

####⚠Avertissement : la version Uwp suspendra les mises à jour des fonctionnalités et entrera dans l'état de maintenance uniquement, RX-Explorer sera migré vers le framework WinAppSdk

>####WinAppSdk est un framework alternatif pour Uwp. RX-Explorer est actuellement refactorisé et migré vers WinAppSdk. La date de lancement spécifique est à déterminer

>####Après le lancement de la version WinAppSdk, la version Uwp sera toujours conservée dans Microsoft Store, et tous les utilisateurs achetés peuvent directement activer la version WinAppSdk
